海外内容运营 AI 面板（合规版）

合规前提：数据都由你手动粘贴（自有评价/公开评论/公开链接），工具只做分析与总结，不做任何“扒取”。 未配置 API Key 时也能用：会输出可复制提示词，你可粘贴到任何大模型对话框继续扩写。

若你从本地文件直接双击打开，遇到“跨域/CORS”导致无法调用 API：用本地服务器打开即可（示例命令见“设置”里的提示）。

输入目标市场与品类，生成可直接拍的 TikTok 脚本拆解与“可喂给 AI 的提示词”。

把海外用户评论/投诉/私信手动粘贴到输入框，一键输出情绪仪表盘、夸赞词云、槽点排序与机会卡片。

输入竞品公开信息（帖子链接、公开评论、或你手动复制的评论区内容），输出“弱点雷达”：情绪概览 + 频繁负面痛点 + 可攻击突破口建议。

让大模型扮演“文化秘书”，为目标国家整理未来一周的公共文化趋势，输出营销灵感日历（不需要抓取任何私有数据）。

🔒 API 调用安全架构

为了保障 API 密钥安全，本项目采用后端中转方案：

  • 前端 → 发送用户请求（不含密钥）
  • Netlify Function → 在后端安全读取环境变量中的密钥
  • 智谱AI → 接收请求并返回结果
  • 前端 → 展示 AI 生成内容

✅ API Key 存储在 Netlify 环境变量中，不会暴露在前端代码里。
